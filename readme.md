# superHeroes



## Prerequisites
- PHP 5.6.31
- Wamp / Xampp

## Database
- Open PhpMyAdmin
- Create a new database named 'hetic_superHeroes_test'
- Go to SQL, copy/paste hetic_superHeroes.sql & execute
- [!] Be careful, you may need to modify the SQL port in includes/config.php. The current SQL port is 3306.
